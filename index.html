How to use this tool
  1. Content Setup: Select the platform, awareness stage, format purpose, and content format. Use Other when the preset list does not fit the assignment.
  2. Content Inputs: Enter the product or brand, audience, key problem or desire, and main benefit. Be specific; the hook and caption will be built from this context.
  3. Hook Builder: Choose a hook type and generate several hooks. Read the rationale, then select the hook that best fits the audience and format.
  4. Output Controls: Set the number of variants, caption length, tone, emoji level, hashtag count, CTA type, and whether an engagement question should be included.
  5. Generate Content: Review the platform-specific outputs. For TikTok, turn on storyboard mode when you need scenes, timing, audio, and Canva or CapCut build notes.
  6. Edit Or Revise: Either edit the selected caption manually or write a short instruction and ask AI to revise it. Do not treat the first draft as final.
  7. Image Asset: Generate image prompts only after the caption is ready. If you change image settings after prompts exist, regenerate prompts or reset the image asset.
  8. Download: Export the caption, storyboard, image prompts, image files, or the full package. Caption and image stay separate; add any text overlays later in Canva or another design tool.
